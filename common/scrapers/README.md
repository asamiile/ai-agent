## common/scrapers/

### 概要
このディレクトリは、様々なECサイトから商品情報やリンクをプログラムで収集するためのスクレイピングモジュール群を格納しています。主に [Playwright](https://playwright.dev/) を使用してブラウザ操作をエミュレートし、ウェブページから特定のデータを抽出します。収集されたデータは、他のエージェント（例: `agents/fashion_diesel_kids`）で利用されることを想定しています。

### ディレクトリ構成

common/scrapers/
├── __init__.py
├── yahoo.py
└── zozo.py


### 各ファイル/モジュールの説明

| ファイル/モジュール名 | 概要                                                                                                                                                                                                                                                                                                                                                            |
| :------------------ | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `__init__.py`       | このディレクトリがPythonパッケージであることを示すための空ファイルです。                                                                                                                                                                                                                                                                                          |
| `yahoo.py`          | **Yahoo!ショッピングのスクレイパー。** <br> Playwrightを使用してYahoo!ショッピングの検索結果ページにアクセスし、「DIESEL KIDS」のような特定のキーワードで検索された商品詳細ページのURLを収集します。複数ページにわたる商品リンクの収集ロジックが含まれています。                                                                                                        |
| `zozo.py`           | **ZOZO.jpのスクレイパー（開発中）。** <br> PlaywrightとBeautifulSoupを組み合わせてZOZO.jpから商品リンクを収集しようと試みるモジュールです。しかし、サイトの自動化検出により「Access Denied」となることが多く、現時点では完全には機能していません。自動化検出を回避するための初期スクリプトが組み込まれています。 |

### 実行方法
`common/scrapers/` ディレクトリ内のスクリプトは、通常、他のメインエージェント（例: `agents/fashion_diesel_kids/main.py`）からインポートされて利用される「部品」としての役割が大きいです。
ただし、各スクレイパーモジュールは単体で実行して動作をテストすることも可能です。

#### `yahoo.py` の単体実行例
Yahoo!ショッピングからDiesel Kids関連の商品リンクを収集します。

bash
python common/scrapers/yahoo.py


#### `zozo.py` の単体実行例
ZOZO.jpからDiesel Kidsブランドの商品リンクを収集しようと試みます。

bash
python common/scrapers/zozo.py


### 環境変数
`common/scrapers/` ディレクトリ内のスクリプト自体は、直接的な環境変数を必要としません。
しかし、依存ライブラリであるPlaywrightを使用するため、以下の設定が必要になる場合があります。

- **Playwrightブラウザのインストール**:
  PlaywrightはChrome、Firefox、WebKitなどのブラウザエンジンを別途ダウンロードして使用します。初めてPlaywrightを使用する際は、以下のコマンドで必要なブラウザをインストールしてください。

    bash
    playwright install
    

- **上位モジュールからの利用**:
  このスクレイパーモジュールを利用する上位のアプリケーションやエージェント（例: `agents/fashion_diesel_kids/main.py`）では、Google AI Studio APIキーやDiscord Webhook URLなどの環境変数を必要とする場合があります。それらの詳細は各エージェントの`README.md`を参照してください。
