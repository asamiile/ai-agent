# `agents/tech_readme_updater/`

## 概要

このディレクトリには、AI (Gemini) を活用してプロジェクト内の README ファイルを自動生成・更新するエージェントが含まれています。プロジェクトルートの README や、特定のサブディレクトリ（特に `agents/` および `common/` 配下で Python ファイルを含むディレクトリ）の README を、そのコードベースと定義されたプロンプトに基づいて作成・更新します。

これにより、ドキュメント作成の手間を削減し、常に最新のコードベースに合致したドキュメントを維持することを目指します。

## ディレクトリ構成


agents/tech_readme_updater/
├── main.py
└── README.md


## 各ファイル/モジュールの説明

| ファイル名      | 説明                                                                                                                                                                                                                                                                                                                                                              |
| :-------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `main.py`       | このエージェントのメインスクリプトです。リポジトリ内のコードを読み込み、Google Gemini API を呼び出して README コンテンツを生成します。生成されたコンテンツは、プロジェクトルートおよび `agents/`、`common/` 配下の Python ファイルを含む各ディレクトリの `README.md` ファイルに書き込まれます。README 生成のためのプロンプト定義も内部に含まれています。 |
| `README.md`     | このディレクトリ（`agents/tech_readme_updater/`）に関する説明ファイルです。                                                                                                                                                                                                                                                                                           |

## 実行方法

1.  **依存ライブラリのインストール**:
    プロジェクトルートの `requirements.txt` に基づいて依存ライブラリをインストールします。

    bash
    pip install -r requirements.txt
    

2.  **環境変数の設定**:
    後述の環境変数を `.env` ファイルに設定するか、環境に直接設定してください。

3.  **スクリプトの実行**:
    プロジェクトルートから以下のコマンドを実行します。これにより、プロジェクトルートと指定されたサブディレクトリの README が更新されます。

    bash
    python agents/tech_readme_updater/main.py
    

    **注意**: `GOOGLE_AI_STUDIO_API_KEY` が設定されていない場合、スクリプトはエラーを発生させずに終了します。これはローカル環境での意図しないAPI呼び出しを防ぐためのものです。

## 環境変数

このエージェントの実行には、以下の環境変数を設定する必要があります。

| 環境変数名                | 説明                                         |
| :------------------------ | :------------------------------------------- |
| `GOOGLE_AI_STUDIO_API_KEY` | Google Gemini API にアクセスするための API キー。 |
